## Recording Tips

1. **Browser**: Chrome, maximized, dark mode
2. **Resolution**: 1920×1080 native (no scaling)
3. **Clear all browser tabs** except TenderShield
4. **Disable notifications** on your system
5. **Pre-login** as Officer first, then use browser back to landing
6. **Practice the flow** 3 times before recording
7. **Record audio** separately in a quiet room, then sync
8. **Background music**: Subtle corporate/tech underscore (royalty-free)
